# Social Launch Briefs

Per-platform notes for launching Repo Swap. Drafts live in `../copy/`, exported
images in `../assets/`.

## The hook (one-liner)

> Change one word in a GitHub URL → unlock 7 superpowers. Repo Swap does it in one click.

## Core message

GitHub repos hide value behind clunky file views. A whole ecosystem of tools
(`gitingest`, `gitmcp`, `gitdiagram`, `deepwiki`, `github.gg`, `stackblitz`,
`star-history`) reuses GitHub's `owner/repo` URL shape. Repo Swap is a tiny,
no-tracking browser extension that swaps you into the right one instantly.

## Platforms

### X / Twitter
- Format: hook tweet + thread (one tool per post, screenshot each).
- CTA: link to landing page; "free, open, works in Chrome/Brave/Arc/Firefox."

### LinkedIn
- Angle: developer productivity / AI-assisted code review.
- Tone: slightly more professional; lead with the time-saved framing.

### Reddit (r/webdev, r/programming, r/chrome_extensions)
- No hard sell. Lead with the *mechanic* ("you can swap the domain on any GitHub
  URL") and mention the extension as a convenience. Disclose you built it.

### Instagram / Threads
- Visual-first: the neobrutalist popup + a before/after of a repo → diagram.
